# AI Interview Assistant

An AI-powered interview preparation platform that helps job seekers practice interviews with personalized, AI-generated questions. The application analyzes a candidate's resume and target job role using Google Gemini AI, then evaluates responses by providing scores, constructive feedback, and sample interview-ready answers.

---

## Features

- 📄 Upload PDF Resume
- 💼 Enter Target Job Role
- 🤖 AI-Generated Interview Questions
- 📝 Personalized Answer Evaluation
- ⭐ AI-Based Scoring
- 💡 Detailed Feedback and Improvement Suggestions
- 📚 Sample Interview-Ready Answers
- 📱 Fully Responsive User Interface
- ☁️ Deployed on Render

---

## How It Works

1. Upload your resume in PDF format.
2. Enter your target job role.
3. The system extracts information from your resume.
4. Google Gemini AI generates personalized interview questions.
5. Answer each interview question.
6. Gemini AI evaluates your responses.
7. Receive:
   - Interview Score
   - Personalized Feedback
   - Improvement Suggestions
   - Sample High-Quality Answers

## Installation

### Clone the repository

```bash
git clone https://github.com/saniya331/AI-Interview-Assistant.git
```

### Move into the project folder

```bash
cd AI-Interview-Assistant
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```
